## 📦 What is RCPComponentDiffFixtures?

RCPComponentDiffFixtures is a collection of test files created to help understand changes in Reality Composer Pro components. It focuses on how Apple's tool stores and manages elements like fields and paths in its USD-based files (specifically USDA format). This makes it easier to analyze updates or differences in component setups.

This tool helps users who work with Reality Composer Pro and want to see how component data changes behind the scenes. It is useful for exploring how objects are built and modified, especially when dealing with complex inspector fields or component side effects.

## 🔍 Why Use This Fixture Collection?

If you are working with Reality Composer Pro and want to learn more about its internal component structure, this fixture set gives you detailed examples to study. Instead of guessing how various changes affect components, you get clear reference files showing differences.
